# 인터랙티브 동화 탐험 웹 애플리케이션

# 사용자 가이드

이 문서는 **일반 사용자**와 **콘텐츠 제작자**가 쉽게 이해하도록, 본 프로그램의 모든 기능과 사용 방법을 설명하는 가이드입니다.

---

## 1. 이 웹페이지는 무엇인가요?

이 웹페이지는 **어린이가 여러 가지 모험 이야기를 탐험할 수 있는 인터랙티브(상호작용형) 그림 이야기 플랫폼**입니다.

그림 속 특정 영역을 클릭하면 새로운 이야기 장면으로 이동하게 되며, 콘텐츠 제작자는 내장된 저작 도구를 통해 새로운 스토리를 쉽게 만들 수 있습니다.

### 핵심 특징

- 📚 **그림 속 탐험**: 어린이가 직관적으로 클릭하여 이야기를 탐험
- 🔄 **쉬운 콘텐츠 관리**: JSON 파일로 스토리 저장/불러오기
- ✏️ **내장 저작 도구**: 프로그래밍 없이 새로운 스토리 제작
- 📱 **반응형 디자인**: 모든 화면 크기에서 정상 동작
- 💾 **로컬 저장**: 서버 없이 브라우저에서 데이터 관리

## 2. 시작하기

### 2.1 프로그램 실행

`index.html` 파일을 웹 브라우저에서 열면 프로그램이 시작됩니다.

### 2.2 대시보드 화면

프로그램을 시작하면 **대시보드**가 표시됩니다. 이곳에서 모든 스토리를 관리합니다.

```
┌─────────────────────────────────────────────────────┐
│  나만의 인터랙티브 동화      [📂 Import] [➕ New]  │
├─────────────────────────────────────────────────────┤
│                                                     │
│  ┌──────────────────────────────────────────────┐  │
│  │ [썸네일] │ 강아지의 행복한 하루              │  │
│  │          │ 에피소드 3개                      │  │
│  │          │ [Play] [Edit] [Download] [Delete] │  │
│  └──────────────────────────────────────────────┘  │
│                                                     │
│  ┌──────────────────────────────────────────────┐  │
│  │ [썸네일] │ 숲 속 모험                        │  │
│  │          │ 에피소드 5개                      │  │
│  │          │ [Play] [Edit] [Download] [Delete] │  │
│  └──────────────────────────────────────────────┘  │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 2.3 대시보드 버튼 설명

| 버튼 | 기능 |
|------|------|
| **📂 Import** | JSON 스토리 파일을 가져와서 라이브러리에 추가 |
| **➕ New** | 새로운 빈 스토리를 만들고 저작 모드로 진입 |
| **Play** | 선택한 스토리를 플레이 모드로 실행 |
| **Edit** | 선택한 스토리를 저작 모드에서 편집 |
| **Download** | 스토리를 JSON 파일로 다운로드 |
| **Delete** | 스토리를 라이브러리에서 삭제 |

## 3. 플레이 모드 (어린이용)

### 3.1 메인 화면

스토리의 **Play** 버튼을 클릭하면 메인 화면이 표시됩니다.

- 전체 화면을 채우는 큰 배경 그림이 보입니다
- 그림 위에는 **클릭 가능한 에피소드 영역**이 있습니다 (마우스를 올리면 강조됨)
- 우측 상단에 **🚪 나가기** 버튼이 있습니다

### 3.2 에피소드 화면

클릭 가능한 영역을 클릭하면 해당 에피소드의 상세 화면으로 이동합니다.

- 에피소드의 그림이 전체 화면에 표시됩니다
- 좌측 상단에 **← 메인으로 돌아가기** 버튼이 있습니다

### 3.3 사용 흐름

```
대시보드 → [Play] → 메인 화면 → [영역 클릭] → 에피소드 화면
                         ↑                          ↓
                         └── [메인으로 돌아가기] ←──┘
                         ↓
                    [나가기] → 대시보드
```

## 4. 저작 모드 (콘텐츠 제작자용)

### 4.1 저작 모드 진입

- **새 스토리**: 대시보드에서 **➕ New** 버튼 클릭
- **기존 스토리 편집**: 스토리 카드에서 **Edit** 버튼 클릭

### 4.2 저작 모드 화면 구성

```
┌─────────────────────────────────────────────────────────────┐
│ [← 나가기] [스토리 제목 입력]  [+ New Episode] [💾 스토리 저장] │
├──────┬────────────────────────────────────┬─────────────────┤
│      │                                    │  에피소드 속성  │
│ 도구 │                                    │                 │
│  ⬜  │       캔버스 영역                  │  이름: [    ]   │
│  ⚪  │     (메인 이미지)                  │  ID:   ep1      │
│  🥚  │                                    │                 │
│  ✏️  │   [에피소드 영역들 표시]           │  이미지: [업로드]│
│  🖌️  │                                    │                 │
│  🧹  │                                    │  영역: (자동)   │
│      │                                    │                 │
│      │                                    │ [Save] [Delete] │
│      │                                    │ [Cancel]        │
└──────┴────────────────────────────────────┴─────────────────┘
```

### 4.3 도구 설명

| 도구 | 이름 | 설명 |
|------|------|------|
| ⬜ | 사각형 | 드래그하여 사각형 영역 생성 |
| ⚪ | 원 | 중심에서 드래그하여 원형 영역 생성 |
| 🥚 | 타원 | 드래그하여 타원형 영역 생성 |
| ✏️ | 자유 경로 | 점을 찍어 자유로운 다각형 영역 생성 |
| 🖌️ | 붓 | Path 영역의 점을 이동하거나 추가 (Path 선택 시만 표시) |
| 🧹 | 지우개 | Path 영역의 점을 삭제 (Path 선택 시만 표시) |

## 5. 새 에피소드 만들기

### 5.1 단계별 가이드

#### Step 1: 메인 이미지 설정

1. 캔버스 영역의 빈 공간을 클릭합니다
2. 파일 선택 창에서 메인 배경 이미지를 선택합니다
3. 이미지가 캔버스에 표시됩니다

#### Step 2: 새 에피소드 생성

1. **+ New Episode** 버튼을 클릭합니다
2. 도구바와 속성 패널이 활성화됩니다

#### Step 3: 에피소드 영역 그리기

**사각형 영역:**
1. ⬜ 도구 선택
2. 캔버스에서 드래그하여 사각형 그리기

**원형 영역:**
1. ⚪ 도구 선택
2. 중심점에서 드래그하여 원 그리기

**타원 영역:**
1. 🥚 도구 선택
2. 드래그하여 타원 그리기

**자유 경로 영역:**
1. ✏️ 도구 선택
2. 원하는 위치를 클릭하여 점 추가
3. 첫 번째 점 근처를 클릭하거나 더블클릭하여 완성

#### Step 4: 에피소드 정보 입력

1. **에피소드 이름**: 표시될 이름 입력 (예: "맛있는 점심")
2. **에피소드 이미지**: 클릭하여 에피소드 상세 이미지 업로드

#### Step 5: 에피소드 저장

1. **Save Episode** 버튼 클릭
2. 에피소드가 스토리에 추가됩니다

### 5.2 유효성 검사

에피소드 저장 시 다음 항목이 필수입니다:
- ✅ 에피소드 이름
- ✅ 에피소드 이미지
- ✅ 진입 영역 (도형)

누락된 항목이 있으면 오류 메시지가 표시됩니다.

## 6. 기존 에피소드 편집하기

### 6.1 에피소드 선택

캔버스에 표시된 에피소드 영역을 클릭하면:
- 해당 에피소드가 선택됩니다 (녹색 테두리로 강조)
- 속성 패널에 에피소드 정보가 표시됩니다
- 도구바가 활성화됩니다

### 6.2 영역 이동 (드래그)

1. 에피소드 영역을 선택합니다
2. 영역 내부를 드래그하여 위치를 변경합니다
3. 변경사항은 자동으로 저장됩니다

### 6.3 영역 크기 조절 (리사이즈)

1. 에피소드 영역을 선택합니다
2. 나타나는 8개의 핸들(모서리, 변 중앙)을 드래그합니다
3. 원하는 크기로 조절합니다

```
┌─[nw]─────[n]─────[ne]─┐
│                       │
[w]      영역 내용      [e]
│                       │
└─[sw]─────[s]─────[se]─┘
```

### 6.4 Path 점 편집

Path(자유 경로) 타입의 영역을 선택하면 추가 도구가 활성화됩니다:

**🖌️ 붓 모드:**
- 점 핸들을 드래그하여 위치 변경
- 선분 중간의 점선 원을 클릭하여 새 점 추가

**🧹 지우개 모드:**
- 점 핸들을 클릭하여 해당 점 삭제
- 최소 3개의 점이 유지되어야 합니다

### 6.5 에피소드 삭제

1. 에피소드 영역을 선택합니다
2. 속성 패널의 **Delete** 버튼을 클릭합니다
3. 확인 메시지에서 **확인**을 클릭합니다

## 7. 스토리 저장 및 내보내기

### 7.1 스토리 저장

1. 편집이 완료되면 **💾 스토리 저장** 버튼을 클릭합니다
2. 스토리가 브라우저의 로컬 저장소에 저장됩니다
3. 새로고침해도 데이터가 유지됩니다

### 7.2 JSON 파일 다운로드

1. 대시보드에서 스토리 카드의 **Download** 버튼을 클릭합니다
2. JSON 파일이 다운로드됩니다
3. 이 파일을 다른 컴퓨터에서 **Import**하여 사용할 수 있습니다

### 7.3 스토리 가져오기

1. 대시보드에서 **📂 Import** 버튼을 클릭합니다
2. JSON 스토리 파일을 선택합니다
3. 동일한 ID의 스토리가 있으면 덮어쓰기 여부를 확인합니다

## 8. 영역 형태 종류

### 8.1 사각형 (Rect)

```
┌──────────────┐
│              │
│   클릭 영역  │
│              │
└──────────────┘
```
- 가장 기본적인 직사각형 영역
- 버튼 형태의 UI에 적합

### 8.2 원 (Circle)

```
      ╭──────╮
     ╱        ╲
    │  클릭   │
    │  영역   │
     ╲        ╱
      ╰──────╯
```
- 정원 형태의 영역
- 화면 비율이 변해도 원형 유지

### 8.3 타원 (Ellipse)

```
    ╭────────────────╮
   ╱                  ╲
  │    클릭 영역      │
   ╲                  ╱
    ╰────────────────╯
```
- 가로/세로 비율이 다른 타원 형태
- 넓은 영역에 적합

### 8.4 자유 경로 (Path)

```
      ╱╲
     ╱  ╲
    ╱    ╲────╮
   │          │
    ╲    ╱───╯
     ╲  ╱
      ╲╱
```
- 점을 연결하여 자유로운 형태 생성
- 복잡한 모양의 영역에 적합
- 붓/지우개로 세밀한 편집 가능

## 9. 이미지 관리

### 9.1 이미지 파일 위치

이미지 파일은 `static/images/` 폴더에 저장하는 것을 권장합니다:

```
project/
├── static/
│   └── images/
│       ├── Main_Yard_Scene.png      # 메인 배경
│       ├── episode_1_lunch.png      # 에피소드 1
│       ├── episode_2_play.png       # 에피소드 2
│       └── episode_3_nap.png        # 에피소드 3
└── story/
    └── my_story.json
```

### 9.2 권장 이미지 사양

| 항목 | 권장 사양 |
|------|----------|
| 형식 | PNG, JPG, WebP |
| 해상도 | 1920x1080 이상 권장 |
| 가로세로비 | 16:9 또는 4:3 권장 |
| 파일 크기 | 2MB 이하 권장 |

### 9.3 주의사항

- 저작 도구에서 업로드한 이미지는 Data URL로 임시 표시됩니다
- 실제 배포 시에는 이미지 파일을 `static/images/` 폴더에 복사하고
- JSON의 경로를 `"static/images/파일명.png"` 형식으로 수정해야 합니다

## 10. JSON 스토리 파일 구조

### 10.1 기본 구조

```json
{
  "storyId": "my-story-001",
  "title": "나의 첫 번째 이야기",
  "mainImage": "static/images/main.png",
  "episodes": [
    {
      "id": "ep1",
      "name": "첫 번째 에피소드",
      "episodeImage": "static/images/episode_1.png",
      "entryRegion": {
        "shape": "rect",
        "coords": { "x": 0.1, "y": 0.2, "width": 0.3, "height": 0.25 }
      }
    }
  ]
}
```

### 10.2 좌표 체계

모든 좌표는 **0~1 사이의 비율값**으로 저장됩니다:
- `0.0` = 왼쪽/위쪽 끝
- `0.5` = 중앙
- `1.0` = 오른쪽/아래쪽 끝

이 방식으로 화면 크기와 관계없이 동일한 비율로 영역이 표시됩니다.

## 11. 문제 해결

### 11.1 스토리가 저장되지 않아요

**원인:** 브라우저 로컬 저장소 용량 초과

**해결:**
1. 불필요한 스토리 삭제
2. 이미지를 외부 파일로 관리 (Data URL 대신 파일 경로 사용)
3. JSON 파일로 다운로드하여 백업

### 11.2 영역이 제대로 표시되지 않아요

**원인:** 좌표 데이터 오류

**해결:**
1. 저작 모드에서 해당 영역 선택
2. 영역을 삭제하고 다시 그리기

### 11.3 Import한 스토리 이미지가 보이지 않아요

**원인:** 이미지 파일 경로 불일치

**해결:**
1. JSON 파일의 이미지 경로 확인
2. 해당 경로에 이미지 파일 존재 여부 확인
3. 경로 수정 또는 이미지 파일 복사

### 11.4 저작 모드에서 나가기가 안 돼요

**원인:** 변경사항 확인 대화상자

**해결:**
1. "저장하지 않은 내용은 사라집니다" 메시지 확인
2. **확인**을 클릭하여 나가기
3. 또는 먼저 **💾 스토리 저장** 후 나가기
