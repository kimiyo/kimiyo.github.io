# 인터랙티브 스토리 플레이어 생성 프롬프트

단일 HTML 파일로 동작하는 그림 탐험 스토리 플레이어를 만들어주세요. 모든 CSS, JavaScript, 스토리 데이터가 하나의 HTML 파일에 포함되어야 하며, 이미지는 `static/images/` 폴더에 저장됩니다.

스토리는 JavaScript 객체로 정의됩니다. 이 객체는 메인 배경 이미지 경로인 `mainImage`, 그리고 메인 이미지의 원본 크기를 픽셀 단위로 저장하는 `imageSize`를 포함합니다. `imageSize`는 클릭 영역의 좌표를 화면에 맞게 변환하는 데 사용됩니다.

에피소드는 `episodes` 배열에 저장되며, 각 에피소드는 클릭 시 보여줄 상세 이미지, 그리고 메인 화면에서 클릭할 수 있는 영역을 정의합니다. `영역의 모양`은 `shape`(rect, circle, ellipse중 하나)와 픽셀 좌표로 구성됩니다.

각 영역의 모든 좌표는 모두 픽셀 단위로 저장됩니다. 

메인 화면에서는 배경 이미지를 화면 중앙에 반응형으로 표시하고, 각 에피소드의 클릭 영역은 투명하게 배치되며, 마우스를 올리면 반투명한 흰색 배경과 글로우 효과로 강조됩니다.

영역을 클릭하면 에피소드 화면으로 전환되며 해당 에피소드의 이미지가 전체 화면에 표시되고, 좌측 상단에는 "← 메인으로 돌아가기" 버튼이 나타납니다.  화면 전환은 0.3초 페이드 애니메이션으로 처리합니다.

반응형 렌더링을 위해 모든 픽셀 좌표는 `(좌표 / imageSize) * 100%` 공식으로 퍼센트 값으로 변환됩니다. 브라우저 창 크기가 변경되면 이 변환을 다시 적용하여 클릭 영역의 위치를 자동으로 재계산합니다. 

사용자가 스토리를 수정하려면 HTML 파일 내의 스토리 객체만 편집하면 됩니다. 이미지 파일들을 `static/images/` 폴더에 저장하고, 메인 이미지의 원본 크기를 확인한 후 `imageSize`에 입력합니다. 그 다음 이미지 편집기에서 원하는 클릭 영역의 좌표를 확인하고 픽셀 값을 `coords`에 입력하면 됩니다.

다음은 스토리 데이터의 예제입니다:

```javascript
const storyData = {
    "mainImage": "static/images/Main_Yard_Scene.png",
    "imageSize": { "width": 1920, "height": 1080 },
    "episodes": [
        {
            "episodeImage": "static/images/episode_1_delicious_lunch.png",
            "entryRegion": {
                "shape": "circle",
                "coords": { "cx": 400, "cy": 700, "r": 120 }
            }
        }
    ]
};
```
